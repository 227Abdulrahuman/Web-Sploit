# Web Sploit 
A bug bounty automation framework that automates enumeration and vulnerability discovery for web applications.

### Subdomain Enumeration Worker 
Enumerates subdomains from:
- Security Trails
- VirusTotal
- Anubis
- Chaos
- Shodan
- alienvault
- certspotter
- bevigil
- urlscan
- digitalyama
- dnsdumpster
- Cisco Umbera.
